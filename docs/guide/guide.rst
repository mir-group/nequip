User Guide
==========

This guide covers using NequIP, from initial setup to advanced optimization techniques.

**Getting Started** provides framework-level information for new users, including installation, basic workflow, and file types.

**Configuration** covers setting up training runs, including config files, data preparation, and model configuration.

**Training Techniques** explores advanced training methods like multitask learning.

**Accelerations** details performance optimization techniques for faster training and inference.

**Reference** contains conventions, troubleshooting guides, and frequently asked questions.
